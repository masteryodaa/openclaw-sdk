# Dashboard

FastAPI application factory for the OpenClaw SDK dashboard. Exposes REST API endpoints
for all SDK features including agents, sessions, config, metrics, webhooks, workflows,
audit, billing, templates, connectors, schedules, and channels.

!!! note "Requires the `dashboard` extra"
    ```bash
    pip install openclaw-sdk[dashboard]
    ```
